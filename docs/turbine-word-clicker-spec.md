# タービン・ワードクリッカー仕様書

## コンセプト

ユーザーが任意の文章を入力すると、その言葉がタービンの燃料になり、中央のタービンが成長して強く回るゲーム。  
テーマ「タービン」を、言葉の流れを回転エネルギーに変える装置として表現する。

## 体験の一文

「言葉を入れるたびに、AI が文章のエネルギーを読み取り、タービンが育って加速する」

## 画面構成

- 画面上部: ユーザー入力フォーム
  - テキスト入力欄
  - 「言葉を流す」ボタン
  - サンプル入力ボタン
- 画面中央: タービン
  - 回転する羽根
  - レベルに応じてサイズ、光、回転速度が変化
  - 入力後に一瞬強く回る演出
- 画面下部: 状態表示
  - タービンレベル
  - 回転力
  - 累計エネルギー
  - AI が判定した言葉の属性
  - 次の成長までのゲージ

## 基本フロー

1. ユーザーが任意の文章を入力する。
2. 「言葉を流す」を押す。
3. AI が文章を分析する。
4. AI がエネルギー量、属性、成長コメントを返す。
5. タービンが加速し、ゲージが増える。
6. 一定量に達するとタービンがレベルアップする。

## AI 入力

```json
{
  "text": "ユーザーが入力した文章",
  "currentLevel": 1,
  "currentEnergy": 20
}
```

## AI 出力

```json
{
  "summary": "入力文の短い要約",
  "energy": 24,
  "type": "発想",
  "reason": "新しい行動につながる言葉が含まれているため",
  "comment": "発想の風がタービンを強く押しました"
}
```

## エネルギー判定

- `energy`: 1 から 50 の整数。
- 短すぎる入力は低め。
- 具体的な行動、感情、発想、決意がある文章は高め。
- ネガティブな文章でも、整理や改善につながる場合はエネルギーになる。

## 属性

AI は入力文を次のいずれかに分類する。

- `発想`: アイデア、企画、ひらめき。
- `感情`: 不安、怒り、喜び、迷い。
- `行動`: やること、決意、計画。
- `混沌`: 散らかったメモ、未整理の考え。
- `静寂`: 短すぎる、意味が薄い、動きが少ない言葉。

## ゲームルール

- 入力するたびに累計エネルギーが増える。
- レベルアップに必要なエネルギーはレベルごとに増える。
- レベルが上がるほどタービンの見た目と回転速度が強くなる。
- 同じ文章を連続入力した場合はエネルギーを下げる。
- 空入力ではエラーを表示し、タービンは回さない。

## レベル演出

| レベル | 見た目 | 回転 |
| --- | --- | --- |
| 1 | 小さなタービン | ゆっくり |
| 2 | 羽根が大きくなる | 少し速い |
| 3 | 周囲に光の輪が出る | 速い |
| 4 | 火花や風の線が出る | かなり速い |
| 5 | 画面全体に風の流れが出る | 最大速度 |

## UI 方針

- 中央のタービンを主役にする。
- 入力フォームは上部に固定し、迷わず使えるようにする。
- AI の存在は、属性、理由、コメントで見せる。
- JSON やモデル名は画面に出さない。
- 文章を入力した瞬間に、画面が変化する気持ちよさを優先する。

## サンプル入力

- 「今日の失敗を明日の改善につなげたい」
- 「新しいサービス案を考えたいけど、まだまとまっていない」
- 「やることが多すぎて何から始めればいいか分からない」

## ローディング

- ボタンを押したらすぐにタービンを軽く回す。
- AI 応答待ち中は「言葉を風に変換中」と表示する。
- 応答後に回転速度とゲージを反映する。

## エラー

- 空入力: 「言葉を入れるとタービンが回ります」
- AI 失敗: 「風が乱れました。もう一度流してください」
- 通信失敗: 「接続が不安定です。サンプル入力で試せます」

## 最小実装

- テキスト入力。
- Gemini API による文章分析。
- タービンの CSS 回転アニメーション。
- エネルギーゲージ。
- レベル表示。
- サンプル入力。
- ローディングとエラー表示。

## 余裕があれば追加

- 入力履歴を風のログとして表示する。
- 属性ごとにタービンの色を変える。
- レベルアップ時に短い称号を出す。
- 一定レベル到達で背景が変化する。
- 30 秒間でどれだけ回せるかのチャレンジモード。

## 3 分デモ

1. 画面中央のタービンを見せる。
2. サンプル入力を押す。
3. 「言葉を流す」を押す。
4. タービンが加速し、ゲージが増えるところを見せる。
5. AI が判定した属性、理由、コメントを見せる。
6. 別の文章を入力して、違う属性と回転変化を見せる。

## 成功条件

- 初見の審査員が、文章を入れるとタービンが育つことを理解できる。
- タービンの成長と回転が気持ちよく見える。
- AI が文章の意味を読み取っていることが結果から分かる。
- 発表なしでも、Chrome で触るだけで遊び方が分かる。
